# Lagrangiana do Zero Field Primordial

## Definição

Campo escalar real mínimo em cosmologia:

$$\mathcal{L} = \frac{1}{2}\partial_\mu\phi\partial^\mu\phi - \frac{1}{2}m^2\phi^2$$

Onde:
- φ: campo escalar real
- m: massa do campo (parâmetro livre, será testado)
- Métrica de fundo: FRW plano (ds² = −dt² + a(t)²(dx² + dy² + dz²))

## Assunções explícitas

1. **Sem Λ**: Nenhuma constante cosmológica; apenas φ como componente de "escuridão".
2. **Sem acoplamentos**: φ não acoplado a matéria ou radiação.
3. **Sem termos quartic**: Apenas potencial V(φ) = ½m²φ².
4. **FRW plano**: Geometria espacial Euclidiana (k = 0).
