# Equação de estado w_φ(z)

## Expressão

$$w_\phi(z) = \frac{\dot{\phi}^2 - m^2\phi^2}{\dot{\phi}^2 + m^2\phi^2}$$

## Limites físicos

- **w_φ → −1**: campo congelado, w → −1 (comportamento tipo Λ).
- **w_φ → 0**: campo oscilando, w → 0 (comportamento tipo matéria).
- **w_φ → 1/3**: campo cinético, w → 1/3 (comportamento tipo radiação).

## Evolução esperada

Tipicamente:
1. Alto z: φ̇² > m²φ² → w_φ ≈ 1/3 (domínio cinético).
2. Transição: φ̇² ↔ m²φ².
3. Baixo z: φ̇² < m²φ² → w_φ → −1 (congelado).
